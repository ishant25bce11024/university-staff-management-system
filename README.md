University Database Management System

Overview of the Project
This project is a command-line application that manages university data, including teachers, students, and other staff members. It uses SQLite for persistent storage and provides separate menu-driven modules for different types of users.

Features
Add, edit, delete, and display teachers.

Promote teachers and view their details by employee ID.

Admit, edit, delete, and display students.

Display students by programme and view details by registration number.

Add, edit, delete, and display non-teaching staff.

Display staff by designation and view details by employee ID.

Clear, menu-driven interface for user inputs and operations.

Technologies/Tools Used
Python 3

SQLite3

Command-line interface

Steps to Install & Run the Project
Ensure Python 3 and SQLite3 are installed on your system.

Download and place all code files (.py) in your directory.

Create a SQLite database file named university.db and define tables (teachers, students, emploies) as per application requirements.

Run the main Python file in the terminal:

text
python maincode.py
Use the menu options to manage data.

Instructions for Testing
Add test data for teachers, students, and staff using the application menus.

Verify that each operation (add, edit, delete, display, promote, filter) works as expected.

Try invalid inputs to check error handling.
